If you’re installing Java on a system running some variety of Linux other than Red Hat, make sure you have installed whatever package includes the libstdc++-libc6.1-1.so.2 shared object file; this is a library that allows C++ code that was compiled with an earlier version of the GNU C++ compiler run on a contemporary system. Sun’s installation instructions tell you to install the compat-libstdc++-6.21-2.9.0.0.i386.rpm package if you’re using Red Hat Advanced Server, but say nothing about the other distributions.

I used to have a much longer posting here describing all the trauma I suffered along the way to learning this fun fact, but typing in the whole story practically bored me to sleep, so there’s no need to subject you to the same misery.
